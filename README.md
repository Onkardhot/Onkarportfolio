## 🚀 How to View

**Option 1: Direct Open**
- Open `C:\Users\UA\portfolio\index.html` in Chrome/Edge

**Option 2: VS Code**
- Install "Live Server" extension
- Right-click index.html → Open with Live Server

**Option 3: Terminal**
```bash
cd portfolio
npx serve
# Open http://localhost:3000
```
